# [윈도우] echo 호스트 추가 

```cmd
$ echo 123.123.12.34 www.asdf.com >> c:\Windows\System32\drivers\etc\hosts
```

IP주소가 123.123.12.34인 사이트 www.asdf.com을 c:\Windows\System32\drivers\etc\hosts 에 추가.
