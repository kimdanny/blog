# vi / vim 단축키 모음

단축키 | 기능
------|------
/{단어} | 검색
n | 검색 이후 다음 해당 단어 위치로 이동
N | 검색 이후 이전 해당 단어 위치로 이동
gg | 문서 가장 상위로 커서 이동
G | 문서 가장 하위로 커서 이동
yy | 복사
p | 붙여넣기
